# MERN RBAC Project

This is a Role-Based Access Control (RBAC) project built with the MERN stack (MongoDB, Express, React, Node.js).

## Setup

1. Clone the repository
2. Run `npm install` in the root directory
3. Run `npm run install` to install dependencies for both frontend and backend
4. Create a `.env` file in the backend directory with your MongoDB URI and JWT secret
5. Run `npm start` to start both the frontend and backend servers

## Features

- User authentication (login, register, password reset)
- Role-based access control
- User management
- Dashboard with statistics

## Technologies Used

- MongoDB
- Express.js
- React
- Node.js
- Redux Toolkit
- JSON Web Tokens
